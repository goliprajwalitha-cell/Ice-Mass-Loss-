# Ice-Mass-Loss-
Ice sheet mass loss analysed using GRACE data.

Utilising the data from GRACE Level-2, produced by the Center for Space Research (CSR) at the University of Texas at Austin, Antarctic Ice Mass loss is mapped using multiple interesting modules.

Ice mass Loss can be described using variations in the Earth's gravitational potential over time and space.  Monthly values of geopotential coefficients are used to quantify changes in gravitational potential between 2020 and 2025.

High level of ice mass loss observed in Amundsen Sea during this time period!
